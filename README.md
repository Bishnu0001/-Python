🐍 Python Basics Repository

This repository contains fundamental Python programs designed to help beginners understand the core concepts of Python programming. The files include examples of variables, data types, operators, collections, functions, lambda expressions, and control flow statements.

The goal of this repository is to provide simple and clear examples that demonstrate how Python works step-by-step. These scripts can be used for learning, revision, or practice for beginners who are starting their journey in Python programming.

📂 Project Structure

The repository contains the following Python files:

File Name	Description
1.0-basic.py	Introduction to Python syntax and basic program structure
1.1-Variables.py	Demonstrates how variables are declared and used in Python
1.2-Datatypes.py	Explains different Python data types such as int, float, string, and boolean
1.3-operators.py	Examples of arithmetic, relational, logical, and assignment operators
3.1.1-ListExamples.py	Practical examples using Python lists
3.1-Lists.py	Introduction to lists and list operations
3.2-Tuples.py	Understanding tuples and their properties
3.3-Sets.py	Demonstrates the usage of sets and set operations
3.4-Dictionaries.py	Examples of dictionary data structures and operations
4.1-functions.py	Basic functions in Python
4.2-examplesfunctions.py	Additional examples of function implementations
4.3-Lambda.py	Introduction to lambda (anonymous) functions
4.4-Mapsfunction.py	Usage of the map() function
4.5-filterfunction.py	Usage of the filter() function
Conditionalstatements.py	Examples of conditional statements (if, elif, else)
Loops.py	Demonstrates loops such as for and while
